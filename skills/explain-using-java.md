Name: explain-using-java

Purpose
-------
Provide mappings and analogies to Java (Spring, common server idioms) for
backend or architectural concepts found in the repository.

When to use
-----------
- The code being explained contains backend patterns (HTTP handlers,
  dependency injection, ORM usage) and the user wants Java parallels.

Behavior
--------
- Map DI/service patterns to Spring services, controllers to REST
  controllers, and show small illustrative snippets where helpful.
