Name: explain-using-react

Purpose
-------
Offer clear, idiomatic React-focused explanations and small example
snippets for code in other frameworks or languages (helpful when the
reader knows Angular/Vue and wants React parallels).

When to use
-----------
- User requests React-centered analogies or minimal React snippets showing
  how a feature would look in React.

Behavior
--------
- Map class-based or other-framework components to React function
  components with hooks, show minimal working snippets, and explain
  lifecycle/state mapping.
