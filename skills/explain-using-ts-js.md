Name: explain-using-ts-js

Purpose
-------
Explain TypeScript and JavaScript idioms, focusing on typing, module
systems, and common runtime considerations. Useful as a base skill for
frontend explanations.

When to use
-----------
- Any file that is `.ts`, `.tsx`, `.js`, or `.jsx` where type or language
  differences matter.

Behavior
--------
- Clarify type annotations, generics, structural typing, and differences
  between TS and JS runtime behavior.
